# Neumorphic Scientific Calculator PWA

This is a Progressive Web App (PWA) version of the Neumorphic Scientific Calculator.  
It includes:
- Scientific calculator with history
- BMI calculator
- Body fat estimator
- Export results to PDF
- Installable as an app on desktop/mobile
- Offline support using Service Worker

## 🚀 Deployment on GitHub Pages
1. Upload all files in this ZIP to a GitHub repository.
2. Go to **Repository Settings > Pages**.
3. Select `main` branch and set it to root (`/`).
4. Open your GitHub Pages link (e.g., `https://username.github.io/Calculator/`).

## 📱 Installation as PWA
- On desktop (Chrome/Edge): Click the **Install App (+ icon)** in the address bar.
- On Android: Open the site in Chrome → Add to Home Screen / Install.
- On iOS: Open in Safari → Share → Add to Home Screen.

Enjoy your new Neumorphic Calculator PWA! 🎉
